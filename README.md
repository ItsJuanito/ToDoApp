  - ~$ pip install Flask
  - ~$ pip install Flask-SQLAlchemy
  - ~$ python app.py

  - bd.sqlite: this file contains the database information and all the attributes.

  - base.html: this file is the main file to run, contains Python formatted scripts and source code.
